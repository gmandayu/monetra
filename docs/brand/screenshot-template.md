# Monetra Screenshot Template

## Purpose

The screenshot template creates consistent portfolio, repository, and release
imagery for Monetra.

The editable reference asset is:

`public/brand/monetra-screenshot-template.svg`

## Canvas

- Default size: 1600 × 900 px.
- Aspect ratio: 16:9.
- Safe area: 72 px from every edge.
- Export format: PNG for sharing and SVG for editable source.

## Layout

1. Place the horizontal Monetra logo in the top-left area.
2. Place a short feature title and optional one-line description above the
   application frame.
3. Place the application screenshot inside the central frame.
4. Keep the bottom-right area available for a release label when needed.

## Screenshot Requirements

- Use realistic but fictional financial data.
- Never expose personal, production, or sensitive information.
- Use a consistent viewport for screenshots in the same series.
- Hide browser extensions, bookmarks, notifications, and unrelated tabs.
- Keep cursor placement away from important content.
- Use the same theme across a single screenshot series.

## Copy

- Feature title: maximum 48 characters.
- Description: maximum 100 characters.
- Use sentence case.
- Avoid marketing claims that the product cannot demonstrate.
- Do not add text directly over the application screenshot.

## Export Naming

```text
monetra-{feature}-{theme}-{size}.png
```

Examples:

```text
monetra-dashboard-light-1600x900.png
monetra-transactions-dark-1600x900.png
monetra-budget-mobile-1600x900.png
```

## Portfolio Set

The recommended portfolio set contains:

1. Dashboard overview.
2. Transaction creation.
3. Account management.
4. Budget progress.
5. Reports and analytics.
6. Responsive mobile view.

## Accessibility

Every published screenshot includes descriptive alternative text. Captions
describe the demonstrated feature instead of repeating visible headings.
