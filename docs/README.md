# Monetra Documentation

This directory contains the product, architecture, engineering, and decision
documentation for Monetra.

Documentation is created and approved in the order defined by the project
`TODO.md`. A document marked as **Draft** may change during product discovery. A
document marked as **Approved** represents the current project decision.

## Product

| Document | Purpose | Status |
| --- | --- | --- |
| [Product Identity](./product/identity.md) | Name, tagline, description, philosophy, and brand personality | Approved |
| Product Vision | Vision, mission, values, goals, and positioning | Pending |
| User Persona | Primary users, needs, behaviors, and constraints | Pending |
| Jobs to Be Done | User situations, motivations, and desired outcomes | Pending |
| Glossary | Shared financial and product terminology | Pending |
| Product Scope | Product boundaries, assumptions, risks, and constraints | Pending |
| Product Roadmap | Planned releases and outcomes | Pending |
| Product Backlog | Prioritized epics, features, and stories | Pending |
